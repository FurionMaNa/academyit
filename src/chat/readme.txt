Программа должна считывать с консоли сообщения неограниченного числа пользователей и
отображать его всем пользователям, которые есть в чате. При первом входе в чат, новый
пользователь должен увидеть всю историю сообщений.

Как пользоваться чатом:
1) Запустить ServerApp
2) Запустить ClientApp
3) Ввести имя пользователя которое будет отображаться в чате
4) Ввести ip адрес где запущен ServerApp
5) Начать переписку
6) Повторять с пункта 2 необходимое число раз
